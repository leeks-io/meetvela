

# VELA — Autonomous Anchor Safety & Audit Agent

**Live Demo / Frontend:** [https://meetvela.vercel.app](https://meetvela.vercel.app)

VELA is an intelligent security assistant for Solana developers. It automatically audits Anchor programs, identifies potential vulnerabilities, and provides actionable guidance to prevent hacks before code goes live. By making security accessible and automated, VELA helps developers build safer dApps and strengthens the Solana ecosystem.

---

## About the Prototype

The first version of VELA was built using **Lovable.dev**, combining AI assistance with on-chain integration. It leverages:

* **Official Solana SDK** for native program interactions
* **Anchor Framework** for secure account validation
* **Rust Documentation** for type-safe, high-performance development
* **GPT-4o / Claude** for intelligent code reasoning

This approach ensures VELA provides **accurate, reliable, and developer-friendly guidance** while remaining fully decentralized and open-source.

---

## Main Pages

* **Frontend Demo:** Interactive web interface at [meetvela.vercel.app](https://meetvela.vercel.app)
* **CLI Tool:** Headless Rust agent (`vela-check`) for local audits and CI/CD integration
* **Open Source Repository:** Complete Rust and frontend code available for contributions and audits

## Open Source & Community

VELA is fully open source and designed for transparency. Developers and community members are encouraged to explore the code, contribute improvements, and help expand adoption across the Solana ecosystem.

## Disclaimer

VELA is a developer productivity and security tool. It provides guidance to improve smart contract safety but does not replace professional audits. Users remain responsible for the deployment of their own programs.
